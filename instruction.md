Одностраничный сайт-портфолио (лендинг) для **дизайнера интерьеров / 3D-визуализатора**.

## 4. Что на работах (анализ по всем 128)
Тип контента — фотореалистичная 3D-визуализация жилых интерьеров, профессиональный уровень.

**Типы помещений (примерно, по убыванию):**
- Гостиные и кухни-гостиные (открытые планировки) — самая большая группа
- Спальни (мастер, подростковые, детские)
- Кухни
- Ванные комнаты и санузлы
- Прихожие, коридоры, холлы с лестницами
- Гардеробные
- Кабинет / домашний офис

**Стили:** современный/контемпорари (основа), скандинавский, японди/минимал,
неоклассика, яркая эклектика (постеры, тропические обои, акцентные стены-фуксия).

**Палитра и настроение:** тёплые нейтральные (greige, бежевый, тёплое дерево),
шалфейно-зелёный, пыльно-розовый; акценты — изумрудный велюр, терракота, латунь/золото,
флористические росписи на стенах. Общее ощущение — тёплый, «насмотренный», уютный минимализм.

## 5. Принятые решения
- Язык сайта — **русский**.
- **Тексты пишет Claude** (имя/контакты/факты подставит владелец позже).
- Формат — **одностраничный лендинг**, акцент на галерею.
- Технологии — **статика без сборки + GSAP**: `index.html` + `css/style.css` + `js/main.js`,
  анимации на GSAP (CDN), без npm-зависимостей. Открывается двойным кликом, кладётся на любой хостинг.
  Адаптив (десктоп/планшет/телефон). Подробности — в `PLAN.md`.
- Изображения нужно **оптимизировать** под веб (исходники тяжёлые): сгенерировать уменьшенные
  версии для сетки + версии покрупнее для просмотра в лайтбоксе.
- **Концепция: «Редакционная студия»** — тёплый минимализм, баланс крупных фото и текста,
  заголовочный шрифт с засечками + чистый гротеск, мягкие анимации при скролле.
- **Объём галереи: кураторская подборка ~40–50 лучших** (дубли и слабые/повторяющиеся кадры убираем).
- **Блоки:** Hero · Галерея (с фильтрами) · Услуги · Этапы работы · Контакты.
  Отдельного «Обо мне» нет — короткое позиционирование уходит в Hero (выбор владельца). Отзывы — позже.

## 6. Концепция (утверждено)
**«Редакционная» подача студии интерьера**: крупные изображения, много воздуха,
элегантный заголовочный шрифт с засечками + чистый гротеск для текста, спокойная тёплая палитра
из самих работ, мягкие анимации при скролле.

Рассмотренные альтернативы (отклонены): «галерея-минимал» и «студийный/продающий».

## 7. Структура страницы (утверждено)
1. **Hero** — крупный кадр, имя/специализация, короткий слоган + позиционирование,
   кнопки «Смотреть работы» / «Связаться».
2. **Портфолио** — фильтруемая галерея по категориям
   (Все / Гостиные / Спальни / Кухни / Ванные / Прихожие …), просмотр в лайтбоксе. **Сердце страницы.**
3. **Услуги** — визуализация, дизайн-проект, авторский надзор и т.п.
4. **Этапы работы** — по шагам (бриф → планировка → визуализация → проект).
5. **Контакты** — призыв + ссылки на мессенджеры/почту/соцсети.
6. **Подвал.**
